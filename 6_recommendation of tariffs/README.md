Классификаиция клиентов телеком компании

Задача: На основе данных предложить клиенту тариф.

Описание: Оператор мобильной связи выяснил: многие клиенты пользуются архивными тарифами. Они хотят построить систему, способную проанализировать поведение клиентов и предложить пользователям один из новых тариф.

Ключи: классификация, подбор гиперпараметров, выбор модели МО
